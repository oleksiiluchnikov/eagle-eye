# Eagle Master

A command line interface for the Eagle App with a focus on automation.
Written in Rust.

## Features

- Format name of the folder

## Installation

```bash
git clone ..
cd eaglemaster
cargo install --path .
```

## Usage

```bash
eaglemaster --help
```

## Development

```bash
git clone ..
cd eaglemaster
cargo run -- --help
```
